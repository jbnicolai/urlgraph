# url graph extracter + visualization

## Fetch dependencies with:
- npm install

## Extract graph with:
- node crawler.js > edges.csv

## Visualize with:
- python -m SimpleHTTPServer
- go to localhost:8000

To test rename edges.csv.example-internal to edges.csv
